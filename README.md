## Install

Substitute ~/.local/bin for a prefered dir the is in your $PATH.

```sh
git clone git@github.com:craftvscruft/tfrefactor.git
cd tfrefactor
go build

cp bin/tfrefactor ~/.local/bin
```

## Usage

```sh
tfrefactor
```

## Run tests

```sh
make test
```
